# Flask Admin PoC

## Install

Minimal instructions to run the project:

```sh
$ python -m venv flask-poc
$ . flask-poc/bin/activate
(flask-poc) $ pip install -e .
(flask-poc) $ python app.py
```

Go to http://127.0.0.1:5000/admin/ and enjoy.

## Utilities

- SQLAlchemy

- Flask-Admin
